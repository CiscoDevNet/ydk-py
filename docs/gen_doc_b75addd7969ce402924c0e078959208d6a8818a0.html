class NetworkInstances.NetworkInstance.Protocols.Protocol.Isis.Interfaces.Interface.Levels.Level.AfiSafi.Af.SegmentRouting.PrefixSids.PrefixSid¶

Bases: Entity

Class Hierarchy : NetworkInstances NetworkInstance Protocols Protocol Isis Interfaces Interface Levels Level AfiSafi Af SegmentRouting PrefixSids PrefixSid

This class represents configuration data.

An IGP prefix that should have a segment routing IGP-Prefix SID

allocated to it. The value of the SID is specified by the SID ID,

as an absolute value. If the absolute value falls within the SRGB,

the Global flag should be advertised by the system.
