class NetworkInstances.NetworkInstance.Protocols.Protocol.Isis.Levels.Level.LinkStateDatabase.Lsp.Tlvs.Tlv.MultiTopology.Topologies.Topology.State.Attributes¶

Attributes (Enum Class)

Attributes of the LSP for the associated

topology.

OVERLOAD = 0

When set, node is overloaded, still part of

the topology but cannot be used for transit.

ATTACHED = 1

When set, node is attached to another area

using the referred metric and can be used as

default gateway.
