class OpticalAmplifier.Amplifiers.Amplifier.State.OutputPowerCBand¶

Bases: Entity

Class Hierarchy : OpticalAmplifier Amplifiers Amplifier State OutputPowerCBand

This class represents state data.

The C band (consisting of approximately 191 to 195 THz or

1530nm to 1565 nm)output optical power of this port in units

of 0.01dBm. If avg/min/max statistics are not supported,

just supply the instant value
