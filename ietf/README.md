# YDK bundle

YDK bundle is a package of python API's that are modeled in YANG.

# Installation

Create a source distribution for the bundle:
```
    $ python setup.py sdist
```

Install the bundle:
```
    $ pip install dist/ydk-*.tar.gz
```
