# YDK

YDK or YANG Development Kit is a Software Development Kit that provides API's that are modeled in YANG.

# Installation

Create a source distribution for YDK:
```
    $ python setup.py sdist
```

Install YDK:
```
    $ pip install dist/ydk-*.tar.gz
```
